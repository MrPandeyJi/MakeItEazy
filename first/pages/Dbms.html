Database Management system

A database management system (DBMS) is a software package designed to define, manipulate, retrieve and manage data in a database. A DBMS generally manipulates the data itself, the data format, field names, record structure and file structure. It also defines rules to validate and manipulate this data. Fourth-generation query languages, such as SQL, are used along with the DBMS package to interact with a database.
